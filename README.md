# 🕹️ Pac-Man Arcade: Vanilla JS Edition

A high-fidelity, arcade-accurate Pac-Man clone built entirely with **HTML5 Canvas** and **Vanilla JavaScript**. No frameworks, no external dependencies—just pure web magic.

## 🚀 Experience the Classic
This project is a labor of love dedicated to recreating the original 1980s arcade experience. From the specific ghost AI behaviors to the pixel-perfect tile collisions, every detail has been tuned for authenticity.

### ✨ Key Features
- **Authentic Ghost AI**:
  - 🔴 **Blinky**: The relentless pursuer who targets you directly.
  - 💗 **Pinky**: The ambusher who tries to get ahead of you.
  - 💎 **Inky**: The unpredictable tactician coordinating with Blinky.
  - 🍊 **Clyde**: The "pokey" ghost who retreats when too close.
- **Classic Mechanics**: Grid-based movement, screen wrapping (tunnels), and multi-phase mode switching (Scatter vs. Chase).
- **Power System**: Eat Power Pellets to turn the tables and devour the ghosts!
- **Persistent High Scores**: Compete against yourself with local-storage-backed high scores.
- **Retro Aesthetics**: Scanline-inspired shadows, pixelated rendering, and a vibrant neon-blue maze.

## 🕹️ Controls
| Action | Key |
| :--- | :--- |
| **Move Up** | `W` or `↑` |
| **Move Down** | `S` or `↓` |
| **Move Left** | `A` or `←` |
| **Move Right** | `D` or `→` |

## 🛠️ Technology Stack
- **Core**: HTML5 Semantic Markup
- **Graphics**: HTML5 Canvas API (2D Context)
- **Logic**: Vanilla JavaScript (ES6+)
- **Styling**: Pure CSS3 with Flexbox layouts
